# Sistem Rekomendasi Produk Berbasis Weighted Score

## Deskripsi Singkat
Aplikasi web berbasis Node.js yang berfungsi untuk memberikan rekomendasi produk terbaik berdasarkan kriteria harga, kualitas, dan stok menggunakan metode Weighted Score.

Aplikasi ini dikembangkan sebagai proyek besar pengganti UAS mata kuliah Logika Pemrograman.

---

## Teknologi yang Digunakan
- Node.js
- Express.js
- EJS (Embedded JavaScript Template)
- Sequelize ORM
- SQLite
- Bootstrap & CSS

---

## Fitur Utama
- Input data produk (CRUD)
- Perhitungan skor rekomendasi menggunakan algoritma Weighted Score
- Tampilan hasil rekomendasi produk
- Implementasi Object-Oriented Programming (OOP)
- Relasi database menggunakan Sequelize
